# My Website Project

## 🚀 Features
- User authentication
- Dashboard
- Backend with Node.js & MongoDB

## 🛠 Tech Stack
- HTML, CSS, JavaScript
- Node.js, Express
- MongoDB

## ▶️ How to Run

### Backend
cd backend
npm install
npm start

### Frontend
Open index.html in browser

## ⚠️ Note
Create a .env file in backend:
MONGO_URI=your_mongo_uri
JWT_SECRET=your_secret
